# my neovim config using lazy.nvim

## features:
- lsp
- fuzzy finder & file browser (nvim-tree & telescope)
- gruvbox & catppuccin (very cool)
- discord presence to show everyone that you use neovim btw.
- statusline (also with themes)
- toggleable terminal
- epic start menu (credits to user '[timteeee' on github](https://github.com/timteeee/))
- color and markdown preview
